# GoTickets

## Versions

Go version: 1.21.4

## Start the database instance

```bash
docker compose up -d
```

## Run the project

```bash
go run src/main.go

```

Server will be running at `http://localhost:3333`

## Build project

```bash
go build src/main.go

```

## Run built

```bash
./main

```

Server will be running at `http://localhost:3333`
